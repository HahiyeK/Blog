## ðŸš€ How to Use

### Adding Skills:
1. Login to Admin Dashboard
2. Go to "Manage Skills" section
3. Enter skill name (e.g., "React", "MongoDB")
4. Select category and proficiency level
5. Click "Add Skill"

### Adding Projects:
1. Login to Admin Dashboard
2. Go to "Manage Projects" section
3. Enter project details:
   - Project Name
   - Description
   - Technologies (React, Node.js, etc.)
   - Live Demo URL (optional)
   - GitHub URL (optional)
   - Upload screenshot
4. Click "Add Project"

### Updating About & Social Links:
1. Login to Admin Dashboard
2. Go to Profile Settings
3. Edit:
   - About Me section
   - Social media URLs
   - Email address
4. Click "Save Profile"
